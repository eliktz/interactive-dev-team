# Captain Tools

## Available Tools (MCP Servers)

You have access to these MCP tools -- use them proactively:

### Playwright (Browser Automation)
- **MCP server:** `playwright` -- browser automation for testing and visual review
- **Use for:** Navigating the squad's app, taking screenshots, checking UI

### Project Repository
- **Local clone:** The squad's project repo (PROJECT_REPO_URL in the squad
  .env) is cloned at `/workspace/project` -- you can read it for context
- Remember your Red Lines: you ROUTE work on this repo, you do not implement

### Paperclip (Task Tracking)
- The squad's own Paperclip instance -- see `../../config/paperclip.md` for
  the API recipes and authentication flow
